<div align="center">
  <h2>Installation</h2>
</div>

To get started with St4ndds-Widgets, clone the repository and install the dependencies.

```bash
# Clone the repository
git clone https://github.com/your-username/st4ndds-widgets.git
cd st4ndds-widgets
```

Install dependencies using npm or yarn:

```bash
# Using npm
npm install

# Using yarn
yarn install
```

After installation, you can integrate the widgets into your project. Refer to the specific widget documentation for usage examples.
