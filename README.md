# St4ndd's Widgets

A small desktop widget project featuring **Clock** and **Spotify** integration, built with **Electron**.  
Widgets can be customized, resized, repositioned, and styled according to your preferences.

---

## Features

- **Clock Widget**
  - Digital, analog, minimal, or fancy design
  - Draggable & sizeable
  - Live time updates
- **Spotify Widget**
  - Displays currently playing tracks via Spotify API
  - Draggable & sizeable
- **Settings Widget**
  - Central control for Clock and Spotify widgets
  - Auto-start on system boot
  - Reset tokens and widget positions
- **Tray Integration**
  - Quick access to Settings
  - Minimize to tray functionality
